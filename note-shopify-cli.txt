- shopify theme pull -h: Hiển thị hướng dẫn lệnh pull theme từ store về local.

- shopify theme pull -e development: Pull theme từ store về local theo config environment "development" trong shopify.theme.toml.

- shopify theme dev -e development: Khởi động dev server local, đồng bộ real-time với store development. Cho phép preview và chỉnh sửa theme qua localhost.

- shopify theme push -e development: Push theme từ local lên store development, ghi đè theme hiện tại trên store.

- shopify theme pull -d: Pull theme từ store về local và xóa các file local không tồn tại trên store (đồng bộ 100%).
